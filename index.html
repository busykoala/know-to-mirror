Know to mirror

Do you know the feeling when you have to configure each and every tool in your new project to use a mirror? This page is an attempt to make your life easier with a collection of "how to mirror" for different tools.

# .npmrc
registry=https://busykoala.io/mirror/
//busykoala.io/mirror/:_auth=TOKEN
//busykoala.io/mirror/:always-auth=true

# .npmrc
registry=https://busykoala.io
_auth=TOKEN
always-auth=true

# /etc/pip.conf
[global]
index=https://busykoala.io/pypi
index-url=https://busykoala.io/simple
trusted-host=busykoala.io

# pyproject.toml
[[tool.poetry.source]]
name = "foo"
url = "https://busykoala.io/simple/"
default = true
